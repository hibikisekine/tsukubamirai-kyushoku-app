## 🚀 次のステップ

### 1. 開発環境のセットアップ

```bash
cd 給食サイト
npm install
```

### 2. 環境変数の設定

`.env.local`ファイルを作成：

```env
ADMIN_PASSWORD=your-secure-password
NEXT_PUBLIC_ADSENSE_CLIENT_ID=ca-pub-xxxxxxxxxxxxx
```

### 3. 開発サーバーの起動

```bash
npm run dev
```

### 4. 初回データのアップロード

1. ブラウザで `http://localhost:3000/admin/upload` にアクセス
2. `data/sample.csv` をアップロード
3. 管理者パスワードを入力

### 5. 本番環境へのデプロイ

詳細は `デプロイ手順.md` を参照してください。
