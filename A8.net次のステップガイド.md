# 🚀 A8.net 次のステップガイド

## 📋 やることリスト

### ステップ1: A8.netで商品リンクを生成する

#### 1-1. A8.netにログイン

1. **A8.netにアクセス**
   - https://www.a8.net/ にアクセス
   - アカウントでログイン

#### 1-2. 広告主を探す

1. **「広告主検索」をクリック**
   - ダッシュボードのメニューから「広告主検索」を選択

2. **給食関連の商品を検索**
   - 検索キーワード例：
     - `ランチボックス`
     - `水筒`
     - `給食袋`
     - `箸箱`
     - `給食マット`
     - `お弁当箱`
     - `キッズ用品`

3. **おすすめの広告主**
   - 無印良品（ランチボックス、水筒）
   - サーモス（水筒）
   - 象印（水筒、ランチボックス）
   - リッチェル（ランチボックス、給食用品）
   - その他、給食関連商品を取り扱う企業

#### 1-3. 商品リンクを生成

1. **広告主を選択**
   - 気になる広告主をクリック

2. **「リンク生成」をクリック**
   - 広告主のページで「リンク生成」ボタンをクリック

3. **商品URLを入力**
   - 広告主のサイトから商品ページのURLをコピー
   - A8.netのリンク生成画面に貼り付け

4. **リンクを生成**
   - 「リンク生成」ボタンをクリック
   - 生成されたA8.netのリンクをコピー

**生成されるリンクの形式**:
```
https://px.a8.net/svt/ejp/?aam2=メディアID&url=商品URL
```

### ステップ2: サイトに商品リンクを追加する

#### 2-1. 商品リンクを準備

給食関連商品のリンクを複数準備：

- **ランチボックス**: 給食用のランチボックス
- **水筒**: 給食用の水筒
- **給食袋**: 給食袋、ランチバッグ
- **箸箱**: 箸箱、箸セット
- **給食マット**: 給食マット、ランチマット

#### 2-2. コードに追加

`app/[date]/page.tsx` の給食関連商品セクションに追加：

```tsx
<div className="flex flex-wrap gap-2">
  <AffiliateLink
    href="https://amzn.to/44d5r5t"
    title="Amazonで給食関連商品"
    className="inline-block px-4 py-2 bg-primary-500 text-white rounded-lg hover:bg-primary-600 transition-colors font-semibold no-underline"
  >
    🛒 Amazonで見る
  </AffiliateLink>
  
  {/* A8.netの商品リンクを追加 */}
  <AffiliateLink
    href="https://px.a8.net/svt/ejp/?aam2=あなたのメディアID&url=商品URL"
    title="ランチボックス"
    className="inline-block px-4 py-2 bg-blue-500 text-white rounded-lg hover:bg-blue-600 transition-colors font-semibold no-underline"
  >
    🍱 ランチボックスを見る
  </AffiliateLink>
  
  <AffiliateLink
    href="https://px.a8.net/svt/ejp/?aam2=あなたのメディアID&url=商品URL"
    title="水筒"
    className="inline-block px-4 py-2 bg-blue-500 text-white rounded-lg hover:bg-blue-600 transition-colors font-semibold no-underline"
  >
    💧 水筒を見る
  </AffiliateLink>
</div>
```

**注意**: 
- `https://px.a8.net/svt/ejp/?aam2=...` の形式のリンクは、そのまま使用できます
- 環境変数 `NEXT_PUBLIC_A8_AFFILIATE_ID` が設定されていれば、通常のURLでも自動変換されます

## 💡 収益化のコツ

### 1. 関連性の高い商品を選ぶ

- 給食サイトなので、給食関連商品が最適
- 季節に合わせた商品（夏は水筒、冬は保温弁当箱など）

### 2. 複数の商品を配置

- 1つの商品だけではなく、複数の選択肢を提供
- ユーザーのニーズに合わせて選べるように

### 3. 自然な配置

- コンテンツの流れを損なわないように配置
- 強引な押し付けは避ける

### 4. 定期的に更新

- 人気商品を定期的に更新
- 季節に合わせた商品に変更
